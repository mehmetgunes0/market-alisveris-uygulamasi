# Market Alışveriş Uygulaması

Bu proje, Python programlama dili kullanılarak geliştirilmiş basit bir market alışveriş simülasyonudur. 
Kullanıcının seçtiği ürünlere göre toplam tutar hesaplanmaktadır.

##  Projenin Amacı

Bu proje aşağıdaki temel programlama konularını pekiştirmek amacıyla geliştirilmiştir:

- Değişken kullanımı
- Koşul yapıları (if-else)
- Döngüler
- Liste yapıları
- Toplam hesaplama mantığı
- Kullanıcıdan veri alma

##  Kullanılan Teknoloji

- Python 3

##  Nasıl Çalıştırılır?

1. Bilgisayarınızda Python 3 yüklü olmalıdır.
2. Proje klasörüne gidin.
3. Terminalde aşağıdaki komutu çalıştırın:

python market_alisverisi_uygulama.py

Bu proje eğitim amaçlı geliştirilmiştir ve temel Python pratiği kazandırmayı hedeflemektedir.
